# Python and Power BI project

Ce projet consiste en une analyse complète du Kaggle Machine Learning & Data Science Survey 2020, une enquête internationale menée auprès de milliers de professionnels et d’étudiants dans les domaines de la data, du machine learning et de l’IA.

L’objectif est de reproduire une démarche réaliste de Data Analyst, depuis le nettoyage des données brutes jusqu’à la création de tableaux de bord exploitables pour la prise de décision.

Objectifs : 
Comprendre le profil des professionnels de la data (rôles, niveaux d’expérience, formations)
Analyser les technologies utilisées (langages, outils, environnements)
Étudier les salaires, secteurs et zones géographiques
Mettre en œuvre un pipeline d’analyse de données complet
Produire des visualisations claires et orientées métier

Ici, l’analyse salariale montre un salaire moyen et médian de 44,5K USD dans la data science. Les rémunérations les plus élevées se situent en Suisse et aux États-Unis, reflétant la maturité des marchés technologiques. Les salaires augmentent avec le niveau d’études et l’expérience, notamment après 10 ans. Les postes les plus techniques et stratégiques sont les mieux rémunérés. La formation et la spécialisation constituent les principaux leviers d’évolution.









![image alt](https://github.com/Zakariya-Data-Analyst/Project/blob/main/slide%203.png?raw=true)

Les résultats montrent une forte standardisation des outils en data science. Tableau et Power BI sont les principaux outils de Business Intelligence. Python s’impose largement comme le langage de référence, suivi de SQL. Côté cloud, Google Cloud Platform et Microsoft Azure concentrent la majorité des usages. Ces technologies constituent aujourd’hui le socle des pratiques data.

Conclusion :
Ce projet met en lumière les dynamiques clés du secteur de la data à partir de l’enquête Kaggle 2020. Les résultats montrent des liens forts entre **formation, expérience, poste et rémunération**, ainsi qu’une **standardisation des outils** autour de Python, SQL et Power BI. Malgré certains biais, l’étude offre une vision pertinente d’un secteur en forte évolution où l’adaptabilité et la montée en compétences sont essentielles.
